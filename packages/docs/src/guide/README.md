# Getting Started

## Installation

Install the `vue-columns` package:

```sh
npm i vue-columns
```

Or

```sh
yarn add vue-columns
```

## Quick start

Import the `Columns` component from `vue-columns`:

```vue{2,6}
<script>
import { Columns } from 'vue-columns'

export default {
  components: {
    Columns,
  },
}
</script>
```
